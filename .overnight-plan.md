## Task 5: Physics Improvements (CODE — DO THIS LAST, AFTER RESEARCH)
**Goal:** Make car physics more realistic

### 5a: Tire Slip Model (lateral + longitudinal)
Current problem: No skid/drift at high speed, turning too easy
Research first (Task 4), then implement:

1. **Commit before changes**: `git commit -am "checkpoint: before tire model improvements"`
2. Add longitudinal slip calculation:
   - `slipRatio = (wheelSpeed - vehicleSpeed) / max(|wheelSpeed|, |vehicleSpeed|, 0.1)`
   - Longitudinal force from slip ratio (Pacejka-like curve)
3. Add proper lateral slip:
   - `slipAngle = atan2(lateralVelocity, longitudinalVelocity)`
   - Lateral force from slip angle (linear region then peak then dropoff)
4. Combine into tire friction circle (total force limited by mu * normalForce)
5. This naturally produces:
   - Understeer (front tires saturate first)
   - Oversteer/drift (rear tires saturate)
   - Speed-dependent turning (high speed = more slip = wider turns)
6. Write tests for slip angle, slip ratio, friction circle
7. Keep existing cornering stiffness as the linear region coefficient
8. Tune: AE86 should be drifty (rear-biased), SEDAN_CAR should understeer

### 5b: Collision Response with Rotation
Current problem: Guardrail hits don't spin the car
1. On collision, compute impact force direction and magnitude
2. Apply both linear impulse AND angular impulse (torque)
   - `angularImpulse = r × F` where r is vector from CG to impact point
   - Update angular velocity (heading change rate)
3. Apply to both guardrail and wall collisions
4. This naturally produces: spin-outs, wall bounces with rotation, realistic crashes
5. Write tests
